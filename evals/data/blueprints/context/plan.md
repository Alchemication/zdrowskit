# Weekly Plan

**Week runs Monday → Sunday.** Target: 3 runs + 2 strength sessions per week (~15 km+ total running).
Day-level schedule is flexible — what matters is hitting the weekly totals by Sunday.
Aim to spread sessions across the week; falling behind by Thursday makes catching up hard.
Prefer weekday sessions (Mon–Fri) to keep weekends free for family.

- Run days: ~5 km each (1 tempo or interval session when fresh, rest easy pace)
- Strength A: Push + core (DB bench press, OHP, dips; progressive overload)
- Strength B: Pull + core (pull-ups, rows, deadlift; progressive overload)
- Prefer not to stack run + strength on the same day
- Rest days: walking, optional core/mobility, usually active with son.

# Diet

Pescatarian. Target 120g+ protein (~1.5 g/kg). No strict calorie tracking.

# Sleep

Target: 6.5–7 hours. Realistically variable with a child.
