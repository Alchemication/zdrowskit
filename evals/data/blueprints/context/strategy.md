# Strategy

## Goals — Current focus (next 4-8 weeks)
- Hold 3 runs + 2 lifts/week at 15+ km volume
- Build tempo capacity: extend tempo segments toward 3 km @ sub-5:10/km
  - Tempo is the #1 weekly priority run — schedule it **by Thursday**; if not done by then, it becomes Friday's session
- Sleep tracking compliance above 70%
- VO2max above 46 (currently 45.44, trending up)

## Goals — Medium-term (3-6 months)
- Sub-25 min 5K (pre-COVID PR; current best pace 5:08/km, need ~5:00/km sustained)
- VO2max above 47
- Consistent 4-run weeks when schedule allows

## Goals — Ongoing
- Build and maintain a strong, lean physique — progress compound lifts while staying injury-free
- Family and health always take priority over numbers

## Weekly Plan

**Week runs Monday → Sunday.** Target: 3 runs + 2 strength sessions per week (~15 km+ total running).
Day-level schedule is flexible — what matters is hitting the weekly totals by Sunday.
Aim to spread sessions across the week; falling behind by Thursday makes catching up hard.
Prefer weekday sessions (Mon–Fri) to keep weekends free for family.

- Run days: ~5 km each
  - 1 tempo session/week when HRV is above personal baseline (~50 ms): tempo segments of 1.5–2.5 km @ 5:05–5:15/km pace
  - Remaining runs at easy pace (5:30–6:00/km)
- Strength A: Push + core (DB bench press, OHP, dips; progressive overload)
- Strength B: Pull + core (pull-ups, rows, deadlift; progressive overload)
- Prefer not to stack run + strength on the same day
- Rest days: walking, optional core/mobility, usually active with son.

## Diet

Pescatarian. Target 120g+ protein (~1.5 g/kg). No strict calorie tracking.

## Sleep

Target: 7+ hours (currently averaging 7.3h). Realistically variable with a child.
Track sleep at least 5 of 7 nights per week.
